# Library Borrowing Tracker

## Overview
The Library Borrowing Tracker is a web application that allows users to manage book borrowing in a library. It provides functionalities for borrowers to log in, view available books, borrow books, and check their borrowing history. Librarians can add new books and manage borrowings.

## Features
- User authentication for borrowers and librarians
- Borrowers can view available books and borrow them
- Librarians can add new books and manage borrowings

## Techstack
- **Frontend**: React, React Router, Axios
- **Backend**: Node.js, Express, MySQL (or your chosen database)
- **Testing**: Jest, React Testing Library
- **Styling**: CSS (or any CSS framework you choose)

## Installation

### Prerequisites
- Node.js
- npm

### Clone the repository
```bash
git clone https://github.com/farrelta/DBProject_Group7_Library-Borrowing-Tracker.git
cd DBProject_Group7_Library-Borrowing-Tracker
```

### Install dependencies
For the frontend:
```bash
cd week4_integration
npm install
```

For the backend:
```bash
cd "week3_CRUD_demo(backend)"
npm install
```

## Running the Application

### Frontend
To start the frontend application:
```bash
cd week4_integration
npm start
```

### Backend
To start the backend application:
```bash
cd week3_CRUD_demo(backend)
npm start
```
